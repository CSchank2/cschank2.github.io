## GitHub Pages Cheatsheet

### Inserting Lists

If you want something like this:

List of hobbies:

* Coding
  * Elm
  * Haskell
    * Stack
    * ghc
  * Markdown
* Biking
* Hiking
* Video Games

Favourite fruits:

1. Strawberries
2. Blueberries
3. Bananas

Use this code:

```markdown
List of hobbies:

* Coding
  * Elm
  * Haskell
    * Stack
    * ghc
* Biking
* Hiking
* Video Games

Favourite fruits:

1. Strawberries
2. Blueberries
3. Bananas
```

### Inserting Links

To make links, like [McMaster Start Coding Website](http://outreach.mcmaster.ca),
use:

```markdown
[McMaster Start Coding Website](http://outreach.mcmaster.ca)
```
