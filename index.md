## GitHub Pages Cheatsheet

### Inserting Lists

If you want something like this:

List of hobbies:

* Coding
  * Elm
  * Haskell
    * Stack
    * ghc
* Biking
* Hiking

Favourite fruits:

1. Strawberries
2. Blueberries
3. Bananas

Use this code:

```markdown
List of hobbies:

* Coding
  * Elm
  * Haskell
    * Stack
    * ghc
* Biking
* Hiking

Favourite fruits:

1. Strawberries
2. Blueberries
3. Bananas
```

### Inserting Links

To make links, like [Outreach Website](http://outreach.mcmaster.ca),
use:

```markdown
[Outreach Website](http://outreach.mcmaster.ca)
```

### Inserting Pictures

![Picture of a brown dog](https://onlydogsandhorses.co.uk/wp-content/uploads/2017/09/DOGS-16-min.jpg)

Find your picture URL on [Yahoo Images](https://images.yahoo.ca), then use:

```markdown
![Picture of a brown dog](https://onlydogsandhorses.co.uk/wp-content/uploads/2017/09/DOGS-16-min.jpg)
```
